# STN-Arbeitsvermittlung
The STN-Arbeitsvermittlung project is a modern web platform dedicated to connecting  clients  with construction or renovation needs and qualified professionals in the building industry.

## Project Launch

````
# 1. Clone the repository
git clone [URL_DU_DEPOT] stn-arbeitsvermittlung
cd stn-arbeitsvermittlung
cd frontend

# 2. Install dependencies
npm install

````
### Run development website

Run the development server using `/tools/run.sh`, then open your browser and go to `http://localhost:3000`.

### Run development server
````
cd backend
````
Run the development server using `/tools/run.sh`, then open your browser and go to `http://localhost:8000`. The default login credentials are username: "root" and password: "root1234".
