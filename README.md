# STN-Arbeitsvermittlung

The STN-Arbeitsvermittlung project is a modern web platform dedicated to connecting clients with construction or renovation needs and qualified professionals in the building industry.

## Project Launch

### Requirements :

Install Git and Docker Desktop.

```
# 1. Clone the repository
git clone [URL_DU_DEPOT] stn-arbeitsvermittlung
cd stn-arbeitsvermittlung
# 2. **Start:** Launch the complete project (Frontend, Backend and DB) with a single command.
docker compose up --build

```

### Run development website

Run the frontend using `docker compose up frontend --build`, then open your browser and go to `http://localhost:3000`.

### Run development server

Run the backend using `docker compose up backend --build`, then open your browser and go to `http://localhost:8000`.
